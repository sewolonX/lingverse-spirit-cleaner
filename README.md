## 📱 接入企业微信通知

手机收通知，随时知道脚本在干嘛。

### 第一步：创建群机器人

1. 打开**企业微信**手机或电脑客户端
2. 进入任意一个内部群（或者新建一个只有你自己的群）
3. 点击群聊右上角 `···` → **消息推送（如果没有的话检查账号是否有消息推送的权限）** → **添加**
4. 给机器人起个名字（如"灵界助手"），点确定
5. 会得到一个 Webhook 地址，格式类似：

```
https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

### 第二步：填入脚本

1. 打开游戏，右下角脚本面板 → **更新** tab
2. 勾选 `☑ 企业微信通知`
3. 把 Webhook 地址粘贴到 **脚本通知** 输入框
4. 点 **测试发送** → 群里收到 "通知测试 OK" 即成功

### 第三步：三通道分流（可选）

如果需要不同类型消息发到不同群：

| 通道 | 用途 |
|---|---|
| 脚本通知 | 清理开始/结束、冥想、死亡、统计简报 |
| 世界消息 | 游戏世界频道聊天转发 |
| 私信 | 师门/道友私聊转发 |

分别创建三个群机器人，填入对应的 Webhook 地址即可。不需要的留空。

### 通知内容

| 事件 | 格式 |
|---|---|
| 🧹 开始清理 | `角色：苏冉` |
| 🔄 清理中(10min) | `神识剩余：120/200` |
| 🧘 开始冥想 | `当前神识 + 预计恢复时间 + 下一轮清理时间` |
| ✨ 高级冥想 | `仙缘 125→123` |
| 💀 角色陨落 | `位置：青云城` |
| 🔄 已引渡归来 | `前往：灵溪村` |
| ✅ 清理结束 | `运行时长/探索次数/遭遇/死亡/修为变化` |
